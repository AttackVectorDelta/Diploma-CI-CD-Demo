Negate
This function negates the matrix.

Type: POST

Path: /api/matrix/negate

Example request:

{ "matrix": { "x1": 1, "x2":2, "x3":3, "x4":4, "y1": 5, "y2":6, "y3":7, "y4":8, "z1": 9, "z2":10, "z3":11, "z4":12, "h1": 13, "h2":14, "h3":15, "h4":16, } }

Params:
matrix - object (required)
- x1 - float (required)
- x2 - float (required)
- x3 - float (required)
- x4 - float (required)
- y1 - float (required)
- y2 - float (required)
- y3 - float (required)
- y4 - float (required)
- z1 - float (required)
- z2 - float (required)
- z3 - float (required)
- z4 - float (required)
- h1 - float (required)
- h2 - float (required)
- h3 - float (required)
- h4 - float (required)
