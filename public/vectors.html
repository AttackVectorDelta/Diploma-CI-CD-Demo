Vector class

On this page you can see examples of all the API functions of the vector class!

Back home

Negate
This function negates the vector.

Type: POST

Path: /api/vectors/negate

Example request:

{"vector": { "x": 1, "y":2, "z":3 }}

Params:
vector - object (required)
- x - float (required)
- y - float (required)
- z - float (required)
- h - float

Add
This function adds two vectors.

Type: POST

Path: /api/vectors/add

Example request:

{"vector1": { "x": 1, "y":2, "z":3 }, "vector2": { "x": 4, "y":5, "z":6 }}

Params:
vector1 - object (required)
- x - float (required)
- y - float (required)
- z - float (required)
- h - float
vector2 - object (required)
- x - float (required)
- y - float (required)
- z - float (required)
- h - float

Path: /api/vectors/scalar

Params:

vector - object (required)
- x - float (required)
- y - float (required)
- z - float (required)
- h - float
